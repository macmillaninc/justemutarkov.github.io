Tutorial how to start with our Emulator

1. go to #releases search for launcher and server, download both(with assembly archive file) and extract in whatever directory you want.

2. go to https://maoci.tk/eft-0.11.7.3333.zip - to download EFT Client (game files in version displayed in archive name)

2.a) extract game files in whatever directory you want

2.b) replacing Assembly-CSharp.dll file: /EscapeFromTarkov_Data/Managed/(here) - (this file is needed for saving progression)

3. start launcher go to Settings tab (launcher should display you whats wrong on the left dragable side)

4. setup Client and Server Location there ( if directory is right errors should disapear with 1 second delay )

6. if you want to login automatickly (directly to mainmenu), go to tab Account and press Login button (first time make sure you do this cause we store password which isnt md5 and game tryies to send is as md5 hash)

7. after login you can change several things there

7.a) appeariance (player model),

7.b) nickname (nickname displayed ingame),

7.c) password to that profile (its text not hashed so you can use like 111 password (its just a placeholder default passwords is password for profile 0 and 1)

7.d) login / email - it dont need to be email you can change it to whatever you want

8. first start server using button Start Server - its very important (cause game will not work properly)

9. after starting server you can start game pressing button Start Game (dont forget to login in the launcher)
