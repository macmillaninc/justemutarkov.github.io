How to Launch a game

Home Screen of the launcher

if you are using local server you dont need to specify anything up there

Settings Screen of the launcher

You need to edit this page as follows

  • Game location - copy paste the folder location where EscapeFromTarkov.exe is located is located into this box
  • Server location - copy paste the folder location where server.exe is located into this box (our Just EmuTarkov)

To Start everything press Start Server first, and wait till it says that it is Listening on port: {port_number}. After this displays press the Start Game button in the launcher and enjoy the Just EmuTarkov Experience
