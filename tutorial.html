How to Launch a game

 

Home Screen of the launcher

if you are using local server you dont need to specify anything up there

 

Settings Screen of the launcher

You need to edit this page as follows
  • Game location - copy paste there folder location where EscapeFromTarkov.exe is located
  • Server location - copy paste there folder location where server.exe is located (our Just EmuTarkov)

To Start everything press Start Server first, and wait till it says that Listening on port: {port_number}. After this displays press Start Game in the launcher and enjoy Just EmuTarkov Experience
